# NgPageLoadingEffert

##主要功能

通过在启动时在`body`注入`div`层,实现全局的动画效果。
动画效果来源于AngularJs的[me-pageloading](https://github.com/Treri/me-pageloading)
与其不同主要在于：

基于最新的Ng4.
同时将操作`SVG`的库由`SnapSvg`切换到`D3.js`。因`SnapSvg`下没调通动画函数。
